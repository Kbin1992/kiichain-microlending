About This Project

This is a Week 2 KiiChain builder project exploring decentralized lending mechanics and PayFi infrastructure. The MicroLending smart contract demonstrates how permissionless finance can work at a protocol level — anyone can deposit collateral, request loans, and repay without intermediaries.

Built on KiiChain Testnet Oro, this project showcases the capabilities of the KiiChain ecosystem for real-world financial use cases. KiiChain is purpose-built for PayFi — bridging traditional finance and decentralized infrastructure.
